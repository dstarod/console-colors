# Colorize your logs!

## Make console output more beautiul!

#### Install:

    pip3 install git+https://github.com/dstarodubtsev/console-colors

#### How to use:

    from console_colors import color
    print(color.red('Red shoes'))

#### Available methods:

- strong
- weak
- underline
- negative
- hidden
- strikethrow
- black
- red
- green
- yellow
- blue
- purple
- cyan
- lightgray
- black_bg
- red_bg
- green_bg
- yellow_bg
- blue_bg
